# 防返贫风险线索文本生成工具 - 完整功能说明

**版本：** 2026-03-24  
**文件：** 风险线索文案生成工具.html

---

## 一、核心功能概述

本工具面向基层村干部，用于生成防返贫风险线索核实情况文本。支持四类风险模板：
1. **医疗支出类**
2. **新增低保类**
3. **住房安全类**
4. **新增残疾类**

---

## 二、完整功能模块

### 模块 1：风险类型选择

**功能：** 用户选择需要处理的风险类型

**选项：**
- 医疗支出类（🏥）
- 新增低保类（💰）
- 住房安全类（🏠）
- 新增残疾类（♿）

**交互：**
- 点击选择后，高亮显示选中项
- 显示"已选择：XXX"提示
- 自动显示对应的输入区域

---

### 模块 2：家庭成员信息管理

#### 2.1 添加家庭成员

**功能：** 动态添加家庭成员，支持添加多人

**字段：**
| 字段 | 类型 | 说明 |
|------|------|------|
| 姓名 | 文本输入 | 必填 |
| 与户主关系 | 文本输入 + 下拉选择 | 支持直接输入或从下拉列表选择 |
| 年龄 | 数字输入 | 0-120 岁 |
| 健康状况 | 下拉选择 | 健康、大病、慢性病、残疾 |
| 是否在校学生 | 下拉选择（25 岁以下显示） | 是/否 |
| 教育阶段 | 下拉选择（在校学生显示） | 义务教育阶段、非义务教育阶段 |
| 学校名称 | 文本输入（非义务教育阶段显示） | 选填 |
| 劳动力情况 | 只读文本框 | 自动生成 |
| 婚育状况 | 下拉选择（可选显示） | 离异、单身、丧偶 |

**关系下拉选项：**
- 户主、配偶、儿子、儿媳、女儿、女婿
- 孙子、孙女、父亲、母亲、兄弟、姐妹、其他

#### 2.2 删除家庭成员

**功能：** 删除已添加的成员

**交互：** 点击成员卡片右上角的"删除"按钮

#### 2.3 自动计算逻辑

**学生身份判定：**
- **<14 岁**：自动认定为义务教育阶段，隐藏学生选择框
- **14-25 岁**：显示"是否在校学生"选择框
- **>25 岁**：隐藏所有学生相关字段

**劳动力情况判定（与户主身份无关）：**

| 年龄段 | 健康状况 | 劳动力类型 | 是否需要手动选择 |
|--------|----------|-----------|----------------|
| <14 岁 | 任意 | 无劳动力 | ❌ |
| 14-25 岁 + 在校 | 任意 | 无劳动力 | ❌ |
| 26-59 岁 | 健康 | 普通劳动力 | ❌ |
| 26-59 岁 | 慢性病/大病/残疾 | 弱劳动力或丧失劳动力 | ✅ 用户手动选择 |
| 60-79 岁 | 健康 | 弱劳动力 | ❌ |
| 60-79 岁 | 慢性病 | 弱劳动力或无劳动力 | ✅ 用户手动选择 |
| 60-79 岁 | 大病/残疾 | 无劳动力 | ❌ |
| ≥80 岁 | 任意 | 无劳动力 | ❌ |

**黄色提示区域：**
- **触发条件：** 需要手动选择且用户未选择
- **显示内容：** 提示文字 + 两个选择按钮
- **点击按钮后：** 自动填写劳动力类型，删除提示区域

**边界提示：**
- 年龄未填写：显示"请输入年龄后，系统将自动判断劳动力情况"
- 年龄=0：同上

### 模块 3：家庭收入信息管理

#### 3.1 务工收入

**功能：** 添加多条务工记录，自动计算合计

**字段：**
| 字段 | 类型 | 说明 |
|------|------|------|
| 务工人员 | 下拉选择 | 从家庭成员中选择 |
| 务工省份 | 下拉选择 | 全国 31 个省市自治区 |
| 务工城市 | 下拉选择 | 根据省份动态加载 |
| 务工县区 | 下拉选择（仅四川省显示） | 四川省内城市才显示 |
| 起始时间 | 月份选择 | 限制：一年前 ~ 当前月份 |
| 截止时间 | 月份选择 | 限制：一年前 ~ 当前月份 |
| 从事行业 | 文本输入 | 例：建筑工地、电子厂 |
| 总收入 | 数字输入 | 自动计算月均收入 |
| 务工时长 | 只读显示 | 自动计算月数 |

**时间约束：**
- 最小值：一年前的月份（例如 2026-03 的前一年是 2025-04）
- 最大值：当前月份
- 手动输入超出范围会自动修正并提示

**低收入提示：**
- **触发条件：** 四川省外务工 + 月薪 < 2000 元
- **弹窗提示：** 显示务工时长、总收入、月均收入
- **提示文字：** "四川省外务工人员月均收入一般不低于 2000 元"

#### 3.2 生产经营收入

**功能：** 添加多条生产经营记录，自动计算净收入

**字段：**
| 字段 | 类型 | 说明 |
|------|------|------|
| 从事人员 | 下拉选择 | 从无务工记录的劳动力中选择 |
| 生产经营方式 | 下拉选择 | 务农、其他 |
| 具体经营方式 | 文本输入（选择"其他"时显示） | 例：经营茶馆、小卖部 |
| 生产经营收入 | 数字输入 | 总收入 |
| 生产经营支出 | 数字输入 | 总成本 |
| 净收入 | 只读显示 | 自动计算（收入 - 支出） |

**低收入检查（务农）：**
- **触发条件：** 选择"务农" + 净收入 > 0
- **计算：** 非务工劳动力人均净收入
- **阈值：** 8500 元
- **弹窗提示：** 超过 8500 元时提示确认

#### 3.3 财产性收入

**功能：** 手动输入财产性收入

**字段：**
| 字段 | 类型 | 说明 |
|------|------|------|
| 财产性收入合计 | 数字输入 | 租金、分红等 |

#### 3.4 转移性收入

**功能：** 添加多条转移性收入记录，自动计算合计

**字段：**
| 字段 | 类型 | 说明 |
|------|------|------|
| 姓名 | 文本输入 | 受益人 |
| 收入类型 | 下拉选择 | 低保金、养老金、残疾人补贴、地力补贴、其他 |
| 具体类型 | 文本输入（选择"其他"时显示） | 例：临时救助金 |
| 金额 | 数字输入 | 自动累加 |

### 模块 9：家庭成员完整性核查

**触发时机：**
- 首次点击"添加务工记录"
- 首次点击"添加生产经营收入"
- 首次点击"添加转移性收入"

**核查对象：**
- 成年人（≥18 岁）
- <60 岁

**核查条件（满足任一）：**
1. ≥35 岁无配偶
2. 有子女（户主有孙辈）但无配偶

**弹窗内容：**
- 标题："家庭成员完整性核查"
- 副标题："以下成员婚育状况存疑"
- 列出所有符合条件的成员
- 每个成员两个按钮：
  - "补充家庭成员"：滚动到家庭成员区域
  - "补充婚育状况"：显示婚育状况选择框
- "跳过（不再提示）"按钮

**注意：** 14 岁以下儿童不参与核查

### 模块 10：收入核实功能

**触发条件：**
- 净收入在 8500-9500 元之间：提示选择"确定"或"再核实"
- 净收入 ≤ 8500 元：强制"再核实"
- 净收入 > 9500 元：直接生成

**核实方式：**
1. **补充家庭成员**：添加缺失的家庭成员
2. **补充婚育状况**：选择离异、单身、丧偶

**存款支付核实：**
- 显示存款来源输入框
- 显示支付金额输入框
- 提供"快捷输入全部金额"按钮

**亲朋代付核实：**
- 添加代付人记录
- 填写：姓名、与病人关系、务工地点、务工收入、代付金额
- 提供"快捷输入"按钮

**文案生成：**
- 原自付支出 - 存款支付/亲朋代付 = 实际自付支出
- 生成特殊格式的核实说明文本

## 三、文案生成规则

### 通用格式

```
经核查，该户 XXX（残疾时间）鉴定为 XX 残疾，为（新导致残疾/补办残疾证）

1、该家庭共同生活人口 X 人，(户主 XXX 年龄岁，健康状况，婚育状况，劳动力情况，配偶 XXX 年龄岁，健康状况，婚育状况，劳动力情况，...)；家庭现有劳动力 X 人。

2、该户收入主要构成由工资性收入：XXX 时间 至 时间 在 省份 城市 县区 从事 行业，扣除必要务工成本后收入 XXX 元，合计 XXX 元；生产经营性收入：XXX 生产经营方式生产经营性净收入 XXX 元，合计净收入 XXX 元；财产性收入 XXX 元；转移性收入 XXX 元（XXX 的 XXX 类型 XXX 元，...）。总收入 XXX 元，人均纯收入 XXX 元。

3、该家庭合规自付支出情况。医疗支出：XXX 时间 因 病种 在 医院 就医，报销后自付 XXX 元；教育支出：XXX 元；后续支出估算：XXX 元。

4、该家庭是否存在"慎重纳入情形"的情况。XXX（仅当选择"有"时显示）
```

### 特殊规则

**健康状况显示：**
- 每个成员都显示健康状况
- 格式：`年龄岁，健康状况，婚育状况，劳动力情况`

**务工收入描述：**
- 格式：`姓名 时间 至 时间 在 省份 城市 县区 从事 行业，扣除必要务工成本后收入 XXX 元`
- 金额：小数点后为 0 时不保留小数（62000 元），有小数时保留 2 位（62000.50 元）

**生产经营收入描述：**
- 格式：`姓名 生产经营方式生产经营性净收入 XXX 元`

**核实情况描述：**
- 有核实时：`医疗支出：XXX，因 XXX 的存款支付 XXX 元。家庭实际自付支出 XXX 元。`
- 或：`医疗支出：XXX，其中 XXX 代付 XXX 元（XXX 于 XXX 务工收入 XXX 元）。家庭实际自付支出 XXX 元。`

**慎重纳入情形：**
- 选择"无"：不显示第 4 条
- 选择"有"：显示第 4 条及详细说明

## 四、技术实现细节

### 4.1 时间约束

**所有月份输入框：**
- 最小值：一年前的月份（`getOneYearAgo()`）
- 最大值：当前月份（`getCurrentYearMonth()`）
- 手动输入超出范围会自动修正并弹窗提示

**计算逻辑：**
- 当前时间：2026 年 3 月
- 一年前：2025 年 4 月（往前推 12 个月，不包含当前月）

### 4.2 自动计算

**务工收入合计：**
- 自动累加所有务工记录的总收入
- 自动计算务工时长（月数）
- 自动计算月均收入（用于低收入检查）

**生产经营收入合计：**
- 自动计算每条记录的净收入（收入 - 支出）
- 自动累加所有记录的净收入
- 自动计算非务工劳动力人均净收入（用于务农检查）

**转移性收入合计：**
- 自动累加所有转移性收入记录的金额

**教育支出显示：**
- 统计非义务教育阶段学生数量
- >0 时显示教育支出输入区域

### 4.3 手动选择保护

**标记机制：**
- 用户手动选择劳动力类型后，添加 `data-manually-set="true"` 标记
- 后续自动计算时，检查标记，不覆盖用户选择
- 年龄或健康状况改变时，清除标记，重新计算

### 4.4 响应式布局

**移动端优化（<768px）：**
- 双列布局自动切换为单列
- 输入框占满整行
- 字体放大到 16px（防止 iOS 缩放）
- 增加内边距，方便触摸

**桌面端（≥768px）：**
- 保持双列布局
- 字体 14px
- 标准内边距

3. **填写收入信息**：
   - 点击"添加务工记录"填写务工信息
   - 点击"添加生产经营收入"填写务农或经营信息
   - 填写财产性收入
   - 点击"添加转移性收入"填写低保金、养老金等
4. **填写支出信息**：
   - 点击"添加医疗支出"填写医疗信息
   - 如有非义务教育阶段学生，填写教育支出
5. **填写慎重纳入情形**：选择"有"或"无"
6. **填写特殊信息**（根据风险类型）：
   - 住房安全类：填写当前家庭住址
   - 残疾类：填写残疾人信息
   - 低保类：填写低保信息
7. **生成文本**：点击"生成文本"按钮
8. **复制文本**：点击"复制文本"按钮

### 注意事项

- 年龄必须填写，否则无法判断劳动力情况
- 务工时间必须在一年前到当前月之间
- 四川省外务工月均收入低于 2000 元会提示
- 务农人均净收入超过 8500 元会提示
- 净收入≤8500 元会强制要求核实
- 14 岁以下儿童不参与婚育核查
